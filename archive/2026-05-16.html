CLARITY Act 15-9通过参议院委员会 + Bitcoin 2026监管峰会——美国加密"清晰时刻"同步引爆，市场却被PPI通胀泼冷水

金融科技 & 监管
监管
CLARITY Act 15-9通过参议院银行委员会，进入全院表决程序
美国参议院银行委员会5月15日以15比9两党票数通过《数字资产市场清晰法案》，是该法案自2025年7月众议院通过后最重要的立法推进。最终版本增至309页，核心妥协点：稳定币奖励可绑定交易/支付行为，但不可对"存款式持有"付利息；CFTC将获大多数数字资产主监管权，SEC继续监管数字证券。下一步需全院60票阈值及与众议院版本协调。参议员Lummis警告：今年若不过，下次机会在2030年。
So What：这是18个月谈判后的里程碑突破。CFTC成主监管方意味着大量现货加密资产将脱离SEC"证券"框架，合规确定性大幅提升，对交易所、机构托管方结构性利好——但60票全院门槛仍是最大变数。
监管
SEC主席Atkins首登Bitcoin 2026大会：宣布"Project Crypto"、链上代币化证券豁免方案数周内落地
SEC主席Paul Atkins出席Bitcoin Las Vegas 2026大会，成为史上首位出席比特币大会的在任SEC主席。他宣布将在数周内发布"创新豁免"方案，允许代币化证券在链上交易；同时推进"Reg Crypto"框架，允许链上代币销售融资。SEC与CFTC已签署谅解备忘录，共同推进"Project Crypto"代币分类指引，结束"证券 vs 商品"双头监管战。FBI局长Patel同场宣布不再将开发者视为打击对象，专注打击外国欺诈。
So What：Atkins公开将前任加密监管定性为"失败"，"执法式监管"时代宣告终结。链上代币化证券豁免一旦落地，将为RWA、证券通证化打开美国市场的合规通道。

TradFi
Charles Schwab正式推出BTC/ETH现货交易（Schwab Crypto，75bps）——$11.77万亿托管资产基地入场
美国最大券商之一Charles Schwab于5月13-14日开始向零售客户分批开放BTC和ETH现货交易，费率75bps，通过独立Schwab Crypto账户运营，Paxos提供执行与子托管，覆盖全美48州（纽约、路易斯安那除外）。Schwab管理$11.77万亿客户资产，现有39.1百万活跃账户，其中已持有约20%美国现货加密ETF。此举直接与Coinbase、Robinhood竞争零售加密交易入口。
So What：Schwab的入场是过去12个月TradFi加密化的最大零售节点。ETF资金同期净流出$233M显示部分资金正从间接产品向直接持仓轮动，Schwab的到来将加速这一趋势。
TradFi
RWA链上总市值突破$314亿，JPMorgan以太坊第二只代币化货币市场基金JLTXX上线
据RWA.xyz截至5月15日数据，RWA链上总市值达$314.2亿，持有者数量同比增6.75%创近期最大单月增幅。本周重要进展：JPMorgan在以太坊推出第二只代币化货币市场基金JLTXX；富达国际首只代币化基金获穆迪最高评级；韩国金融委员会宣布7月发布代币证券细则；Binance Research预测2030年全球加密用户将达30亿。
So What：RWA赛道已从"概念验证"进入"竞相量产"阶段——JPMorgan开第二只、富达获最高评级、监管细则同步出炉，以太坊作为机构RWA承载层的地位持续强化。

市场
美国4月PPI同比6%创2022年新高 + 油价破$100，BTC ETF单周流出$8.3亿，市场全线杀跌12万人爆仓
美国4月PPI同比涨6%、CPI涨3.8%，均超预期，美联储年内加息概率升至44%（CME FedWatch），美伊冲突致油价破$100/桶。美国10年期国债收益率升至4.54%（年内最高）。BTC单周ETF流出$8.3亿，5/15单日净流出$2.9亿（IBIT $1.36亿领跌），ETH ETF连续5日失血$6,570万。全市场24小时12万人爆仓$4.06亿，多单爆仓占逾90%。BTC从本周高点$82K跌破$79K，7天内回撤约4%。
So What：这是七周连续流入后的首次重大反转。通胀数据直接重定价降息预期，加密与纳斯达克同步走弱，印证BTC与风险资产的高相关性——CLARITY利好已被宏观完全对冲。
市场
CME + ICE向CFTC施压要求监管Hyperliquid：链上石油永续合约"可能扭曲全球油价"，HYPE跌6%
据Bloomberg报道，芝加哥商品交易所（CME）和洲际交易所（ICE）联合向CFTC及国会山施压，要求对去中心化衍生品平台Hyperliquid展开调查。两大交易所指控Hyperliquid的匿名链上石油永续合约市场存在价格操纵和制裁规避风险。Hyperliquid今年3月WTI石油合约单日交易量曾超$12亿，美伊冲突期间比CME市场早48小时反映油价波动，被传统交易所视为"定价权威胁"。HYPE代币应声下跌约6%至$43.81。
So What：这是DeFi衍生品与传统商品市场的第一次正面冲突——Hyperliquid的24/7链上流动性已大到足以影响实体商品价格发现。CME/ICE的监管施压本质是既得利益者的防御反应，但也揭示DeFi合规化迫在眉睫。

DeFi
THORChain遭跨链攻击损失约$1,080万，攻击者提前借Monero-Hyperliquid路径预洗钱
5月15日，跨链流动性协议THORChain遭受多链同步攻击，攻击者通过"Vault Churn地址毒化"漏洞，跨BTC、ETH、BNB Chain、Base等9条链窃取约$1,080万。ZachXBT及Chainalysis追踪发现，攻击者早在数周前即通过Monero隐私桥和Hyperliquid隐藏资金并预建节点质押，攻击前5小时仍有资金转移。协议已触发全局紧急暂停，RUNE代币跌约14%至$0.50。Blockaid指出修复补丁因自动更新系统故障未能及时部署。
So What：这是THORChain的第三次重大安全事故，其累计损失正逼近$2,500万。更值得警惕的是Monero+Hyperliquid洗钱路径——这一组合正在被专业攻击者系统化利用，对整个跨链基础设施的合规压力形成双重打击。

Kevin Warsh正式确认为美联储下任主席（54-45），Powell任期5/15届满
美国参议院以54-45票确认Kevin Warsh为下任美联储主席，鲍威尔任期于5月15日到期。Warsh历史上以鹰派立场著称，市场关注其对现有货币政策路径的态度——在当前PPI 6%、CPI 3.8%的通胀背景下，其正式就任或进一步强化加息预期，压制加密与风险资产。
So What：Warsh接班是5月宏观逆风中的关键变量。加密市场已在本周将美联储加息预期重新定价——若Warsh首次公开发言偏鹰，将进一步压制BTC突破$82K阻力位的时间窗口。
