CLARITY 法案 Markup 倒计时 48 小时：参院银行委员会周四 10:30 ET 正式表决，通过概率 50:50

美国参议院银行委员会已正式排期本周四（5/14）美东时间 10:30 对《数字资产市场 CLARITY 法案》进行 Markup——这是本届国会最关键的加密监管节点。Tillis-Alsobrooks 稳定币收益妥协案已清障，但共和党内 John Kennedy 立场未定、银行业协会周末加码游说；Coinbase 连续三日在华盛顿召开合规圆桌，敦促议员推动全文表决。AInvest 给出 50:50 通过概率，若过会则 SEC / CFTC 管辖权划分有望年内闭环，这将是 GENIUS 法案之后美国加密立法最大一跃。

Consensys 正式致函 SEC：现行 DeFi 解释框架在自托管钱包处留有漏洞

Consensys 法务负责人 Bill Hughes 5 月 11 日向 SEC 提交意见函，指 SEC 上月就加密资产证券定性发布的解释性公告与幕僚声明，在"自托管钱包服务商"这一类别上留下模糊地带，MetaMask 等钱包可能被迫承担超出 DeFi 接口的合规义务。建议 SEC 在 CLARITY 法案配套规则中明确"非托管钱包非经纪商"豁免。

So What：SEC Atkins 时代的"政策化监管"刚开张就被主流 DeFi 基础设施商"反向递意见"——说明链上钱包行业正在从"被动合规"转向"主动立法博弈"。若钱包豁免落不了地，DeFi 前端一侧会被迫 KYC，这是对 MetaMask / Rabby / Phantom 等用户入口级产品的直接威胁。
监管

Tillis-Alsobrooks 妥协文本完整披露：稳定币"被动利息"全面禁止、"平台奖励"保留

CryptoBriefing 拿到参议院银行委员会 5/14 Markup 最新版草案：禁止发行人向持有者支付任何"经济等同存款利息"的被动收益；但允许交易所/钱包/平台基于"真实支付或交易活动"发放与平台激励挂钩的奖励（如 Coinbase USDC 奖励）。妥协文本以"收益路径"而非"收益率"划线，回避了银行业此前 4% 硬上限主张。

So What：Coinbase / Circle 的"USDC 奖励"商业模式保住核心；但 Ondo / Maker 等"DeFi 结构化美债收益类稳定币"将被迫剥离"接近利息"的派息表述，短期内收益型稳定币品类增长可能放缓。
监管

SEC 新 DeFi 接口指引出炉：Uniswap / 以太坊 L2 明确不构成经纪商

SEC 在 CLARITY 法案 Markup 前夕发布 DeFi 前端解释指引：确认非托管 DeFi 接口（前端网站、智能合约调用）不构成证券法意义上的经纪自营商；但强调链上流动性挖矿、平台质押奖励仍需个案分析。UNI、ETH 及主流 L2 代币在本次指引中被明确排除在"证券发行"定义外。

So What：这是 SEC Peirce-Atkins 班底第一份对 DeFi 友好的实质性文件，意味着"执法即监管"路线正式翻篇。对以太坊生态是明显利好——前端不担责 = DEX 长尾项目合规路径明朗，L2 叙事重回监管正名轨道。

AWS AgentCore 细节曝光：Base + Solana 双链，x402 协议锁定 AI 机器经济结算层

Amazon Web Services 本月与 Coinbase、Stripe 联合推出 AgentCore Payments，允许 AI Agent 使用 USDC 稳定币直接购买 API / 数据 / 在线服务。Decrypt 本周披露技术栈细节：底层走 Base + Solana 双链；支付协议采用 Coinbase 主导、现由 Linux Foundation 托管的 x402。上周 Solana × Google Cloud pay.sh 首发后，AWS 版本让三大云中两家在一周内完成 AI Agent 稳定币支付能力对齐。

So What：机器经济结算层（Machine-to-Machine Payments）的技术路线已全面收敛到"稳定币 + x402 + Base/Solana"组合，Azure 补齐只是时间问题。这意味着下一代 SaaS 定价范式——"按 API 调用计费、按 Token 结算"——将通过云厂商自上而下普及，稳定币支付场景占比有望从当前 5% 翻倍。
AI

Solana × Google Cloud pay.sh 落地：AI Agent 按 API 请求付费跳过传统账号体系

Solana 基金会与 Google Cloud 联合发布 pay.sh：AI Agent 可按"单次 API 请求"粒度向 Google Cloud 和社区 API 付费，完全绕过传统云账户注册/信用卡绑定流程。该服务同样基于 x402 协议，链上结算走 Solana 主网，费用以 USDC 计价。

So What：AI Agent 不再需要"拥有信用卡/银行账户"这一传统金融身份——这是机器经济独立于人类金融基础设施的第一个量产案例。对开发者的意义：API 定价从"每月订阅"直接跳到"每次调用"，长尾 API 商业模式被重写。
