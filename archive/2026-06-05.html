双重投降信号：BTC ETF连续12日净流出$39.7亿创历史最长，短期持有者53,800枚BTC低价割肉
据 CoinShares 和 CoinQuant 数据，美国现货比特币ETF已连续12个交易日净流出，累计赎回 $39.7 亿，创历史第二大规模。与此同时，CryptoQuant 记录到过去24小时短期持有者（近期在$80K附近买入）向交易所转移 53,800 枚BTC，且全部以亏损方式进行——为今年最强烈的"投降信号"。BTC 一度触及 $61,348，较5月$82K高点累计回调22%，加密总市值本月已蒸发约$1900亿。全网27.6万人爆仓，金额逾$15亿。
↗ So What：机构与散户同时甩卖是底部形成的前置信号，但结构性压力（AI吸金$4000亿 + 加息预期升温）仍未消散——双重投降信号意味着极端情绪，历史上往往是阶段性底部，但需等待催化剂确认。

Saylor：$4000亿AI融资蚕食BTC流动性——这是资本轮换，不是比特币信仰崩塌
Strategy 董事长 Michael Saylor 在6月4日推文中将近期BTC下跌归因于AI基础设施建设过去6个月吸走约 $4000 亿资金。BTC ETF 自5月14日起净流出约 $40 亿。目前 Strategy 持有 843,706 枚BTC，账面浮亏约 $110 亿，但未见减持迹象，公司同步扩张 Mosaic 企业AI平台。
↗ So What：Saylor 给市场提供了一套叙事框架——BTC跌不是需求消失，而是资本暂时被AI虹吸。若AI投资潮退（或SpaceX IPO后流动性释放），资金可能回流。
TradFi
CoinShares：机构Q1减持BTC ETF 17%，对冲基金砍仓39%，唯独银行逆势翻倍加仓
CoinShares 今日发布Q1机构持仓分析：专业投资者整体减持约 52,000 枚BTC（从313,000降至261,000），对冲基金和券商分别砍仓39%和53%，投顾减持5.9%。但银行机构反向操作，增持 7,800 枚BTC，持仓翻超一倍。Q1 BTC跌幅约22%，一度跌破 $60K。
↗ So What：机构持仓分化耐人寻味——高频交易型机构离场，而受监管的银行反而在抄底。CLARITY法案推进将加速银行合规持仓，长线结构性买方正在积累筹码。
监管 & 立法
监管
白宫数字资产顾问Witt：CLARITY Act"支持监管、支持执法"，今夏窗口期不容错过
白宫数字资产顾问 Patrick Witt 今日称《CLARITY Act》是"支持监管、支持执法"的法案，直接回应执法机构对反洗钱力度的批评，并强调立法者已在法案中新增执法专章。参议员 Lummis 警告：若今年不通过，可能要等到2030年。目前法案已进入参议院立法日历 Calendar No.423，预测市场 Polymarket 给出55%通过概率，Kalshi 给出38%（2027年前）。
↗ So What：白宫直接下场背书执法条款，消除了DeFi支持者与执法机构之间的最大分歧，7月4日全院投票目标更加明确。通过概率低于50%仍反映高度不确定性——民主党7票是关键变量。
监管
【今日新进展】CLARITY Act进入参议院立法日历No.423，本周国会重开后进入全院博弈关键期
《数字资产市场清晰法案》已于6月1日正式列入参议院立法日历 General Orders No.423，6月3日国会复会后进入全院投票程序。支持者目标8月国会夏休前完成表决，但仍需凑齐至少7位民主党议员跨越60票程序性门槛。DeFi 非托管开发者豁免条款已纳入参议院版本。
↗ So What：从委员会到全院立法日历是实质性推进——这是法案"过审"后最关键的程序节点。未来焦点转向民主党7票、利益冲突披露条款措辞，以及两院版本协调。
基础设施 & 衍生品创新
基础设施
Coinbase 推出 SpaceX Pre-IPO 永续合约（USDC结算），6/12 IPO倒计时 · Binance/Kalshi已先行入场
Coinbase（6月4日）推出全新 Pre-IPO 永续期货品类，首发标的为 SpaceX（代码 SPCX-PERP），USDC结算、24小时交易、最高5倍杠杆、上市后自动转换为标准永续合约。目前仅对美国以外合格投资者开放，由百慕大 Coinbase Bermuda 提供。SpaceX 计划6月12日在纳斯达克以 $135/股上市，目标融资 $750亿，估值约 $1.75万亿。此前 Binance、Hyperliquid、Kalshi 已相继推出类似产品。
↗ So What：Pre-IPO 永续合约是链上金融向传统私募市场"价格发现"功能渗透的标志性产品——加密交易所开始与传统IPO承销商争夺一级市场定价权，同时为熊市中的平台提供多元化收入。
监管
Kalshi 获CFTC许可在美首批上线永续合约，Pre-IPO衍生品监管先例正式确立
预测市场 Kalshi 宣布获得美国 CFTC 许可，推出美国境内首批合规永续合约产品，成为继 Coinbase Bermuda（离岸）之后，在本土监管框架下落地 Pre-IPO 衍生品的首家机构。这标志着美国境内加密衍生品监管边界正在从期货扩展至永续产品类别。
↗ So What：CFTC 批准 Kalshi 永续合约为这一新品类提供了监管先例，与 CLARITY Act 推进同向共振——监管框架的扩展速度正在超过预期，金融科技衍生品创新窗口打开。

英伟达 GTC 台北发布 Cosmos 3：全球首款完全开放全模态物理AI模型，训练周期从数月压缩至数天
英伟达在台北GTC大会（6月5日）发布 Cosmos 3，基于混合 Transformer（MoT）架构，整合视觉推理、世界生成与动作预测三大能力，支持文本/图像/视频/环境音/动作多模态原生生成。Cosmos 3 Super 和 Nano 即日可用，物理AI训练评估周期从数月压缩至数天。英伟达同步宣布成立 Cosmos Coalition 全球开发者协作联盟（含 Agile Robots、Runway 等6家创始成员）。理想汽车是首批部署方之一。
↗ So What：Cosmos 3 是AI算力需求的新增量故事——机器人+自动驾驶数据飞轮正式启动，英伟达GTC宣发日博通Q2业绩失望（-12.6%），AI基础设施市场格局重排，英伟达与博通的分化将延续。
