今日头条

AWS × Coinbase × Stripe 推出 AgentCore Payments —— AI Agent 用稳定币自主付款时代正式开启

亚马逊云于 5 月 7 日正式发布 AgentCore Payments 预览版，AI Agent 可使用 USDC 在 Base 与 Solana 上直接支付 API、数据、MCP 服务、甚至其他 Agent。采用 Coinbase 的 x402 协议，由 Stripe 旗下 Privy 提供钱包基础设施。这是全球最大云厂商首次将加密微支付嵌入开发工具链 —— "AI 经济"的第一条清算轨道落地。

▎AI + 金融科技
AI × 支付

AWS 发布 AgentCore Payments：AI Agent 用 USDC 自主支付

Amazon Bedrock AgentCore Payments 预览版正式上线，允许 AI Agent 在任务执行过程中为 API、网页内容、MCP 服务器、其他 Agent 等数字服务付费。底层采用 Coinbase 的 x402 协议 + USDC（Base/Solana），钱包由 Stripe 收购的 Privy 提供。未来将扩展至酒店预订、差旅支付、商户付款。
So What：全球最大云平台首次将加密微支付直接嵌入开发者工具链 —— 企业不再需要自建合规支付基础设施。x402 协议从行业标准变成"云原生"原语，稳定币首次获得超越交易/结算的 B 端开发者场景。AI Agent 经济的基础设施拼图基本集齐。
AI × 支付

RedotPay × Tempo：AI Agent 稳定币支付在 Stripe 背书 L1 上线

跨境支付公司 RedotPay 宣布接入 Tempo Blockchain（Stripe + Paradigm 支持的 L1），采用 Tempo 的 Machine Payments Protocol (MPP)，让自主 AI Agent 执行日常交易。与 AWS/Coinbase 方案形成多链并行生态，Tempo 路线更侧重机器间 M2M 支付语义。
So What：Stripe 在加密支付上双线下注 —— 一端与 AWS/Coinbase 合作 Privy+USDC，一端支持 Paradigm 押注的新型 L1。AI Agent 支付不是单一赢家市场，协议标准之争才刚开始。
▎监管 & 立法
监管

参议院银行委员会锁定 5 月 14 日对 CLARITY 法案进行 Markup

多方消息证实，参议院银行委员会将最早于 5 月 14 日对《数字资产市场清晰法案》进行 Markup 投票，立法草案已向选定行业参与者分发。主席 Tim Scott 希望在 5 月 21 日阵亡将士纪念日休会前完成。Polymarket 对该法案 2026 年内通过的概率上调至 70%。
So What：与上月通过的 GENIUS Act（稳定币）共同构成美国加密监管的"双法案框架"。一旦 Markup 通过，SEC/CFTC 管辖划分、市场结构规则将首次法律化。Coinbase、Circle、Robinhood 的长期政策折价将被重估。
监管

SEC 主席 Atkins：将为链上市场与 AI 驱动金融制定新规

SEC 主席 Paul Atkins 于 5 月 8 日表态，鉴于金融业正被区块链与 AI 双重重塑，SEC 正考虑为链上交易系统、加密金库、区块链结算基础设施等推出新规则制定。将扩大经纪交易商、交易系统等市场参与者的合规边界。
So What：SEC 从"执法优先"转向"规则制定优先"的路径确认 —— 这与过去 Gensler 时代的监管风格彻底切割。与 AgentCore Payments 同日的表态意味深长：监管与技术落地正在同频推进。
▎传统金融 × 数字资产
TradFi

DTCC 7 月启动代币化证券试点：贝莱德、摩根大通、高盛等 50+ 机构参与

管理 $114 万亿资产的 DTCC 确认 2026 年 7 月启动代币化证券有限生产交易，10 月正式推出服务。首批涵盖 Russell 1000 成分股、主要 ETF 与美债，基于 Composer-X 平台，代币化美债部分将部署在 Canton Network。参与机构包括贝莱德、摩根大通、高盛、美银、嘉信、富国、纳斯达克、纽交所、Citadel、Robinhood、Circle、Kraken。
So What：美国证券清算的"中枢神经系统"正式拥抱代币化。Russell 1000 + ETF + 美债意味着 $70 万亿以上资产可能进入链上流通框架。这是比 GENIUS/CLARITY 法案更直接的"华尔街链改"。
TradFi

Meta × Stripe：向创作者发放 USDC，Solana/Polygon 自托管钱包直达

Meta 以 Stripe 为支付服务商，允许哥伦比亚与菲律宾的 Instagram/Facebook/WhatsApp 创作者以 USDC 直接领取收益，支持 MetaMask、Phantom、Binance 钱包，跑在 Solana 或 Polygon 上。自 2022 年 Diem 项目下线后，Meta 首次重返加密支付 —— 这一次没有自己的代币、没有联盟链，只是使用现成的 USDC + 自托管钱包。
So What：30 亿用户级超级平台的支付基础设施选择了"现成稳定币 + 自托管钱包"的最小路径 —— 这对 Diem/Libra 式"平台自发币"的路线画上句号。真正的赢家是 Circle 与公链支付层。
TradFi

Bullish 以 $42 亿收购 Equiniti：代币化证券基础设施的一次大洗牌

Consensus Miami 首日宣布，上市交易所运营商 Bullish 以 $42 亿现金+股票收购英国股东管理/过户代理巨头 Equiniti，以构建"传统证券过户代理 × 代币化基础设施"合一的平台。Kraken × MoneyGram 同日宣布全球 100+ 国 crypto-to-cash 网络合作。
So What：代币化证券的瓶颈从来不是链，而是过户代理、股东名册、企业行为处理等"老骨头"业务。Bullish 跳过自建，直接吞并传统基础设施商 —— 这与 Broadridge 进入链上代理投票同一逻辑：TradFi 的管道正被收购并链改。
▎市场 & 资产结构
市场

Bitwise CEO：四年周期结束，稳定币供应突破 $3000 亿

Bitwise CEO Hunter Horsley 在 Consensus 2026 演讲中宣称传统"三年涨一年跌"的四年周期已失效 —— 去年市场下跌打破规律，机构化参与和广泛采用已改变周期性。同期稳定币总供应突破 $3000 亿（含收益型代币）。
So What：周期论失效意味着 BTC/ETH 的贝塔属性下降、基本面属性上升 —— 未来定价更依赖利率、ETF 资金流、稳定币净铸造等宏观变量。$3000 亿稳定币是"加密金融新M1"，比 BTC 总市值更重要的流动性指标。
市场

5 月 7 日加密市场全线跳水（BTC 跌破 $80K，超 10 万人爆仓、爆仓总额 $3.41 亿、多单占比 74%），今日随欧美股市情绪回暖、美伊临时协议仍在磋商中，BTC 反弹至 $80,366 (+0.80%)、ETH 回到 $2,315 (+1.32%)。恐惧贪婪指数仍处 38（Fear），结构性去杠杆后市场观望情绪浓厚。
So What：本轮回调没有伴随 ETF 显著流出，更像是周期顶部的健康去杠杆。Fear 区域 + 机构长期资金仍在 + 稳定币供应持续扩张 —— 多空分歧在"短期动量 vs 结构性买盘"。
RWA

RWA 链上总市值稳站 $309 亿，稳定币反弹至 $3000 亿上方

本周 RWA 上链资产总市值达 $309 亿，稳定币总市值重回 $3000 亿以上。但月度转账量骤降 25.68%，创近期最大跌幅，显示大额结算与套利需求萎缩。月活地址与持有者数量平稳增长，市场从"高频交易驱动"转向"配置性持有与场景渗透并重"。Centrifuge、Balcony 等 RWA 项目完成新一轮融资。
So What：RWA 第一波"美债代币化"红利已释放完毕，进入"场景渗透 + 资产多样化"第二阶段。转账量下降不是坏消息 —— 这是从"炒作"转向"真实持有"的结构性转变。

AI Agent × 稳定币 × 监管同频 —— 数字资产从"金融操作系统"升级为"机器经济结算层"

本周的三大叙事在同一时间窗口汇聚：① AI Agent 支付基础设施落地（AWS AgentCore + x402 + USDC / RedotPay + Tempo MPP）—— 加密支付首次获得云原生 B 端开发者入口；② 美国监管双支柱成型（GENIUS 已过 + CLARITY 锁定 5/14 Markup + SEC 表态为链上 & AI 金融制规）—— 合规轨道为机构资金扫清最后障碍；③ 传统金融基础设施链改加速（DTCC 7 月试点 + Meta USDC 支付 + Bullish 收购 Equiniti）—— TradFi 的"管道层"正被直接嫁接到链上。三条线共同指向一个更底层的变化：数字资产的下一轮增长动力，不再是加密原生用户，而是 AI Agent + 传统机构 + 合规资金构成的"机器经济"。这意味着估值逻辑从"比特币 ETF 资金流"扩展到"AI Agent 消耗稳定币的速率"，从"山寨币叙事"扩展到"链上基础设施费率"。
